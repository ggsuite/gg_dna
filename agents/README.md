# agents

Konfigurationen für KI-Agenten, die mit `gg_dna` arbeiten — Skills, Settings, Hooks, Agent-Definitionen, Prompt-Bausteine.

Inhalte hier sind dafür gedacht, in Claude-Code-Setups eingebunden oder direkt nach `~/.claude/` bzw. `.claude/` eines Zielprojekts kopiert zu werden. `gg_dna sync` spiegelt diesen Ordner als `dna/agents/` ins Zielprojekt; `install-skills` und `apply-conventions` kopieren von dort weiter nach `.claude/skills/` bzw. `.claude/conventions/`.
